# Pipeline Outputs

This directory contains all generated outputs from the ETL pipeline.

## Output Structure

```
outputs/
├── validation_report.txt              # Pandas validation results
├── 01_sales_over_time.png            # Time series visualizations
├── 02_sales_by_store_type.png        # Store type analysis
├── 03_sales_by_store_size.png        # Store size analysis
├── 04_sales_by_type_and_size.png     # Combined analysis
├── summary_statistics.csv             # Descriptive statistics
└── pyspark/
    ├── pyspark_validation_report.txt  # PySpark validation results
    └── processed_data_sample.csv      # Sample of cleaned data
```

## Regenerating Outputs

All outputs can be regenerated by running the notebooks:
1. `01_eda_exploration.ipynb` - Generates pandas outputs
2. `02_eda_pysparkk.ipynb` - Generates PySpark outputs

**Note**: Output files are not tracked in version control as they can be reproduced from source data.
